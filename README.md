## 🚀 Sobre mim

Sou um **Backend Engineer** com foco em **Node.js, TypeScript, Laravel, PostgreSQL e AWS**.  
Atuo projetando **APIs escaláveis**, sistemas **SaaS multi-tenant**, pipelines de **CI/CD** e ambientes produtivos em **VPS KVM da Hostinger** com **Coolify**.

Tenho experiência com:

- arquitetura backend orientada a serviços

- observabilidade com CloudWatch
- deploy e operação de aplicações em produção
